This tutorial will teach you how to use SASS in your project.

How to start?

1. Prepare the environment

You can check this tutorial on how to setup environment.

1.1. Text editor

Download and install Visual Studio Code

1.2 Browser

Download and install one of the following browser:

  • Google Chrome
  • Firefox
  • Opera

1.3 Lesson files

Download or clone lesson files into your computer.

You will see the following files

        tutorial/ 
            1-start-here.html (current file)
            2-installation.html
            3-variables-maps.html
            4-nesting.html
            5-functions.html
            6-imports.html
        final/

2. Start learning

Navigate to tutorial folder and follow lesson files in given order:

  1. 1-start-here.html (current file)
  2. 2-installation.html
  3. 3-variables-maps.html
  4. 4-nesting.html
  5. 5-functions.html
  6. 6-imports.html
