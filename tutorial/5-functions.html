Functions

One of the biggest advantages of SASS are functions, which like in normal programming languages can accept argument and return us values:

Let's define function to replace our map-get with custom function returning color: 


    @function color($color-name) {
        @return map-get($colors, $color-name)
    }

and replace current invocation: 


    background-color: map-get($colors, background);

with new function call:


    background-color: color(background);

As you can see there is no change in our project so our function works as expected.

Calculations

SASS allows us to do a calculation like below:


    @function my-calculation-function($some-number, $another-number){
        @return $some-number + $another-number
      }

Mixins

Often we are using a bunch of properties together like the one below: 


    display: flex;
    justify-content: center;
    align-items: center;

We can create so called @mixin


    @mixin flexAlign {
        display: flex;
        justify-content: center;
        align-items: center;
    }

and invoke it using @include:


    .subtitle{
        @include flexAlign();
        font-size: 30px;
    }

@mixins with arguments

Mixin can also accept arguments:


    @mixin flexAlign($background) {
        display: flex;
        justify-content: center;
        align-items: center;
        background-color: $background;
    }

which we can invoke as follows:


    .subtitle{
        @include flexAlign(white);
        font-size: 30px;
    }

We can also specify a default value after : sign - now we don't have to always pass it to our mixin call:


    @mixin flexAlign($background: inherit)
